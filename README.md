# goserve

serve static html, support basepath

## use

### install

```sh
go install github.com/gofulljs/goserve
```

### serve resource

current docs

gopher2 will show

```sh
goserve docs
# also use serve via npm install -g serve
# serve docs
```

gopher show

```sh
goserve docs -p goserve
```

this demo at https://gofulljs.github.io/goserve/
